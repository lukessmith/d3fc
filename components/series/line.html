Line Series View source on GitHub

The line series renders the given data as a line, constructed from an SVG path:

var line = fc.series.line()
    .xScale(xScale)
    .yScale(yScale);

container.append('g')
    .datum(data)
    .call(line);

Which gives the following:

This series has the same properties for specifying the x and y value accessors, and decorate as the point series.
